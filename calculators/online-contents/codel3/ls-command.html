ls Command in Linux

The ls command is used to list files and directories in the Linux file system. It’s one of the most frequently used commands to view the contents of a directory.

Basic Syntax

ls [options] [directory]

Common Examples

  • List files in current directory: ls
  • List detailed information: ls -l
  • Include hidden files: ls -a
  • Sort by modification time: ls -lt
  • List files in a specific directory: ls /etc

Useful Options

  • -l: Long listing format
  • -a: Include hidden files
  • -h: Human-readable file sizes (use with -l)
  • -t: Sort by modification time
  • -R: Recursive listing of directories

Example

$ ls -lh
-rw-r--r-- 1 user user 2.1K Jun 24 10:30 notes.txt
drwxr-xr-x 2 user user 4.0K Jun 23 08:15 projects

Tips

  • Use ls -lah to get a complete, human-readable view of all files including hidden ones.
  • Combine options like ls -ltr to sort by time in reverse order.
  • Use alias ll='ls -lh' to shorten common combinations.
