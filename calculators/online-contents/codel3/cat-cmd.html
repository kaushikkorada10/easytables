cat Command in Linux

The cat command in Linux is used to display, combine, and create text files. It's short for ‚Äúconcatenate,‚Äù and is one of the most frequently used commands for working with file contents directly from the terminal.

Basic Syntax

cat [options] [filename(s)]

Common Uses

  • View file content: cat file.txt
  • Concatenate files: cat file1.txt file2.txt > merged.txt
  • Create a file: cat > newfile.txt (press Ctrl+D to save)
  • Append to file: cat >> file.txt
  • Show line numbers: cat -n file.txt

Useful Options

  • -n : Number all lines
  • -b : Number non-empty lines only
  • -s : Squeeze multiple blank lines
  • -E : Show $ at the end of each line

Example

$ cat notes.txt
Linux is fun!
Learning commands.

$ cat -n notes.txt
     1  Linux is fun!
     2  Learning commands.

Best Practices

Use cat for quick reading and file concatenation. For long files, tools like less or more may be better for scrolling. Always redirect output to avoid overwriting unintentionally.
