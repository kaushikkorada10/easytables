pwd Command in Linux

The pwd command stands for "print working directory". It displays the full path of the current directory you are working in. It’s a simple yet essential command used frequently in Linux terminal sessions.

Basic Syntax

pwd [options]

Common Examples

  • Show current directory path: pwd
  • Show physical directory (resolving symlinks): pwd -P
  • Show logical path (default): pwd -L

Useful Options

  • -L: Use PWD from environment, even if it contains symbolic links (default)
  • -P: Avoid symlinks and print the actual physical directory

Example

$ pwd
/home/user/projects

$ cd ../Documents
$ pwd
/home/user/Documents

Tips

  • Use pwd to confirm your location before using commands like cp, mv, or rm.
  • Use pwd -P in scripts to avoid issues with symbolic links.
  • It's often used in combination with scripting and automation to ensure file paths are correct.
