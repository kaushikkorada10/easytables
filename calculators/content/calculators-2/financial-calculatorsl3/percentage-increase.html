What is Percentage Increase?

Percentage increase is the amount of growth between an original value and a new value, expressed as a percentage. It shows how much larger the new number is compared to the starting value. It's commonly used in finance, education, retail, and health-related comparisons.

Percentage Increase Formula

Percentage Increase = [(New Value − Original Value) ÷ Original Value] × 100

Example:

If your sales increased from ₹50,000 to ₹65,000:

  • Difference = ₹65,000 − ₹50,000 = ₹15,000
  • Percentage Increase = (15,000 ÷ 50,000) × 100 = 30%

So, your sales increased by 30%.

Percentage Increase vs Absolute Increase

  • Absolute Increase: Just the difference between the new and original value (e.g., ₹20,000 − ₹15,000 = ₹5,000)
  • Percentage Increase: Expresses the increase relative to the original (e.g., ₹5,000 is 33.33% of ₹15,000)

Common Errors in Calculating Percentage Increase

  • ❌ Swapping new and original values — always subtract the original from the new
  • ❌ Dividing by the wrong base — always divide by the original value
  • ❌ Forgetting to multiply by 100 — percentage is a relative measure
  • ❌ Using the percentage value instead of percentage points in analytics reports

Benefits Over Absolute Value Analysis

Absolute increase shows the raw difference, but percentage increase shows the growth relative to size. This is important for comparing performance between a small company and a large one or a new store vs a flagship outlet.

Example:

  • Small brand: ₹10,000 → ₹15,000 = 50% growth
  • Big brand: ₹1 Cr → ₹1.1 Cr = 10% growth

Though the bigger brand added more rupees, the smaller one had higher percentage growth.

Percentage Increase vs Percentage Change

While both terms are similar, "percentage change" includes both increase and decrease, whereas "percentage increase" only refers to positive growth.

Example:

  • Old: 100, New: 130 → 30% increase
  • Old: 100, New: 70 → 30% decrease (not an increase)

For scenarios where values drop, use the Percentage Decrease Calculator.
