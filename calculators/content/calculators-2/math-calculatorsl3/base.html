üî¢ What is a Number Base?

A **number base** (also called radix) defines the number of unique digits, including zero, used to represent numbers in a positional numeral system. The most common bases include:

  • Binary (Base 2): Uses digits 0 and 1
  • Octal (Base 8): Uses digits 0 to 7
  • Decimal (Base 10): Uses digits 0 to 9 (most commonly used in daily life)
  • Hexadecimal (Base 16): Uses digits 0 to 9 and letters A to F

Each base is important in various applications ranging from data encoding to low-level hardware programming.

üìö Understanding Hexadecimal (Base 16)

Hexadecimal is especially common in computing due to its compact representation of binary values. One hex digit equals four binary digits (bits). For example:
Hex: A = Binary: 1010

A byte (8 bits) can be represented as two hex digits. For example:
Binary: 11111111 ‚Üí Hex: FF

That‚Äôs why you‚Äôll often see memory addresses and color values written in hex.
