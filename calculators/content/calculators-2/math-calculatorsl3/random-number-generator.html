What Is a Random Number Generator (RNG)?

A random number generator is a system or algorithm that produces numbers which cannot be predicted logically and appear completely by chance. In computing, there are two main types:

  • True Random Number Generator (TRNG): Based on unpredictable physical phenomena such as atmospheric noise.
  • Pseudo-Random Number Generator (PRNG): Uses algorithms to generate sequences that mimic randomness but are deterministic.
