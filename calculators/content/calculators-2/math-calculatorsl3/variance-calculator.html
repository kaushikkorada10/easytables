What Is Variance?

Variance measures the average squared deviation of each number in a dataset from the mean of that dataset. It quantifies how much individual data points differ from the average value.

Formula for Population Variance (σ²):

σ² = ∑(x - μ)² / N

  • x = each value
  • μ = population mean
  • N = number of values in the population

Formula for Sample Variance (s²):

s² = ∑(x - ẏ)² / (n - 1)

  • x = each value
  • ẏ = sample mean
  • n = number of values in the sample

Population vs. Sample Variance

Use population variance when your dataset represents the entire group you're analyzing. Use sample variance when your data is a subset of the larger population.

Variance vs. Standard Deviation

Standard deviation is simply the square root of variance. While variance gives you the average squared deviation, standard deviation tells you how much spread exists in original units. Both are used interchangeably in many analyses, but standard deviation is often preferred for interpretation.

Step-by-Step Breakdown of Variance Calculation

Let’s walk through a more detailed step-by-step process of calculating sample variance to help solidify your understanding:

  1. Find the Mean (Average):
    Sum all data points and divide by the number of entries.
    Example: For the dataset 5, 7, 3, 7, 9 — Mean = (5+7+3+7+9)/5 = 6.2
  2. Find the Deviation of Each Data Point from the Mean:
    Subtract the mean from each value.
    Results: -1.2, 0.8, -3.2, 0.8, 2.8
  3. Square Each Deviation:
    1.44, 0.64, 10.24, 0.64, 7.84
  4. Add the Squared Deviations:
    Sum = 1.44 + 0.64 + 10.24 + 0.64 + 7.84 = 20.8
  5. Divide by (n - 1) for Sample Variance:
    20.8 / (5 - 1) = 5.2 → Sample Variance = 5.2

In the field of machine learning, variance plays a crucial role in understanding model performance. High variance indicates that the model is overfitting—capturing noise rather than signal. Low variance models may underfit the data. Balancing bias and variance is a core concept in supervised learning.

Understanding Squared Units

Because variance uses squared deviations, its units are the square of the original data’s units. For instance:

  • If your data is in kilograms, the variance will be in kg².
  • If your data is in meters, the variance will be in m².

This is why many people prefer using standard deviation, which returns the result to original units.

Limitations of Variance

  • Variance is sensitive to outliers — a single extreme value can distort the result.
  • Hard to interpret in real-world terms due to squared units.
  • Cannot be negative, so it doesn't show direction of deviation.
  • May not reflect true spread in small datasets.
